# Agents

This directory contains custom agents for specialized tasks.

## Overview
Agents are specialized prompts or tools that help Claude perform specific tasks more effectively.

## Creating a New Agent
1. Create a new markdown file in this directory
2. Name it descriptively (e.g., `code-reviewer.md`, `test-generator.md`)
3. Define the agent's purpose, capabilities, and instructions

## Example Agent Structure
```markdown
# Agent Name

## Purpose
[What this agent does]

## Capabilities
- [Capability 1]
- [Capability 2]

## Instructions
[Detailed instructions for the agent]
```

## Available Agents
- **planner** - Strategic planning and task breakdown specialist
- **checker** - Quality assurance and code review specialist
- **backend** - FastAPI and Python backend specialist
- **frontend** - React and TypeScript frontend specialist
